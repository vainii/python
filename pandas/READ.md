Projetos para praticar manipulação de dados com Pandas e utilizando da extensão Jupyter, simulando operações comuns em SQL:
- JOIN
- GROUP BY
- ORDER BY
- WHERE

Os dados foram gerados artificialmente através do Chat GPT e contêm inconsistências reais:
- valores nulos
- duplicidades
- dados incompletos

Técnicas utilizadas:
- pandas (merge, groupby, fillna, dropna, etc)
